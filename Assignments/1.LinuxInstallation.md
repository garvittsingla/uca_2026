### 5. Java Setup
1. Install Java in your Linux distribution/environment.
2. Create a file named `HelloUCA.java` that prints `"Hello, UCA 2026!"`.
3. Compile and run the file.

```java
// HelloUCA.java
public class HelloUCA {
    public static void main(String[] args) {
        System.out.println("Hello, UCA 2026!");
    }
}
```

```bash
# Compilation
javac HelloUCA.java

# Execution
java HelloUCA
```

4. Check-in the file to your GitHub repository.
