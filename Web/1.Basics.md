## HTML
1. index.html is the default entering point for websites 
2. ```<!DOCTYPE html>``` Tell the browesr what is the current version of html, now it is just doctype but in 1990, if it was not written browser would change completely how the website looks
3. ```<html> ``` This is the root element of every HTML document and we can give metadata of language which is english to tell browser what is the language of website
4. ```<meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0">``` , utf is the character set, and telling browser what is the character set used for the website , utf - 8 is the most famous and widely used around the globe. Treat the viewport width as the actual width of the user's device.     

> The way in which the files are written, the browser will download sequentially, so it is recommneded to put first style and script.js 
> It is a common thing like we can make it defer
> defer = "Download now, execute later (after the HTML is ready)."
> async = "Download now, execute as soon as possible, even if it interrupts HTML parsing."
